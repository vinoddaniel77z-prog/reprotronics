📖 Setup Guide — No Coding Required
💡 One-time setup in 10 minutes. After that, adding staff takes 2 minutes per person. QR codes work forever — completely free, no subscriptions.
Step-by-Step Setup
1

Create a free GitHub account

Go to github.com and sign up free. Remember your username.

2

Create a new repository

Click + → "New Repository". Name it reprotronics. Set to Public. Check "Add a README file". Click "Create".

3

Enable GitHub Pages

In your repository → Settings → Pages. Source: Deploy from a branch → main. Click Save. Your site: https://YOURUSERNAME.github.io/reprotronics

4

Enter your GitHub URL in Company Info

Paste your URL into the GitHub Pages Base URL field on the Company Info tab. Save.

5

Upload your logo and fill company details

Complete all company details including logo. Write your About Us text using the rich editor. Save company info.

6

Add staff members

Go to Staff Cards → Add Staff Member. Fill in details, upload photo, save.

7

Export and upload profile pages

Click Export All as ZIP. Extract the ZIP. Upload all HTML files to your GitHub repository via "Add file" → "Upload files".

8

Download QR codes

Click 📱 QR Code on any staff card → Download QR PNG. Print on business cards, badges, or materials.

9

Editing later — always Save Admin File

After any changes, click 💾 Save Admin File. This bakes ALL data into a new admin.html. Open that file next time — everything is already there.
